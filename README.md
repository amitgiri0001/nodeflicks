# Nodeflicks
  The application pull images from flickr public feeds and shows in grid format in UI. It has a Search that can fetch images on the basis of searched tags.
  
 # How to run
  npm install
  cd public 
  bower install
  
  npm start
  
  # How to test
    ## Test whole app
      npm test
    ## Test just client
      npm run test-client
    ## Test just server
      npm run test-server
