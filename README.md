# Nodeflicks
  The application pull images from flickr public feeds and shows in grid format in UI. It has a Search that can fetch images on the basis of searched tags.
 
 # Installation
    npm install
    cd public
    bower install  
  
 # How to run
    npm start
  
  # How to test
    npm test
    
  ### Test just client
    npm run test-client
  ### Test just server
    npm run test-server
